# Hospital Management Dashboard

## Project Overview

This project presents an interactive Hospital Management Dashboard developed in Power BI using a dataset containing 10,000 hospital records. The dashboard provides insights into patient admissions, department performance, billing information, and overall hospital operations through interactive visualizations.

## Dataset

* Dataset: Hospital Management Dataset
* Records: 10,000
* Format: CSV

## Data Preparation

Data cleaning and transformation were performed using Power Query, including:

* Removing duplicate records
* Handling missing values
* Correcting data types
* Standardizing text fields
* Removing unnecessary columns

## Dashboard Features

* Patient Admission Analysis
* Department-wise Performance Overview
* Billing and Revenue Analysis
* Interactive Charts and Visualizations
* Filters and Slicers for Dynamic Reporting

## Tools Used

* Power BI Desktop
* Power Query
* CSV Dataset

## Project Files

* `hospital_management.pbix` – Power BI Dashboard
* `Hospital_Management_10k_Records.csv` – Source Dataset
* `README.md` – Project Documentation

## How to Use

1. Download or clone this repository.
2. Open `hospital_management.pbix` using Power BI Desktop.
3. Refresh the data if required.
4. Explore the dashboard using the available filters and visualizations.
